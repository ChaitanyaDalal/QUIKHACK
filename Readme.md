QuickShare is a minimal, no-login, temporary text sharing tool built for fast code/output transfer between two systems.

It is designed specifically for hackathons or short-term usage where:
- WhatsApp is unreliable
- Telegram breaks formatting
- Git is unnecessary for quick snippet sharing

This tool provides a single shared text buffer accessible via one URL.

---

## Features

- Single shared text area
- Send / Overwrite functionality
- Manual Refresh (F5 supported)
- Delete button
- Auto delete after 5 minutes
- No authentication
- No database
- In-memory storage only

---

## Tech Stack

- Node.js
- Express
- Plain HTML + CSS + Vanilla JS
- In-memory variable storage (no database)
